# 📊 E-Commerce Sales Dashboard (Excel Project)

## 🔍 Overview

This project presents an interactive **E-Commerce Sales Dashboard** built using Microsoft Excel. It provides insights into sales performance across different dimensions such as sales channels, brands, industries, and sales representatives.

The dashboard is designed to help stakeholders quickly understand key trends and make data-driven decisions.

---

## 🚀 Key Features

* 📅 **Dynamic Filters (Slicers)**

  * Month
  * Year
  * Region
* 📈 **Sales Rep Performance Analysis**
* 🏷️ **Brand Performance Comparison**
* 🏭 **Industry-wise Sales Insights**
* 🛒 **Sales Channel Distribution**
* 🔄 Fully interactive and user-friendly

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Visualization Techniques

---

## 📊 Dashboard Components

1. **Sales Rep Performance**

   * Identifies top-performing sales representatives
   * Helps track individual contributions

2. **Brand Performance**

   * Compares revenue generated by different brands

3. **Industry Performance**

   * Highlights industries driving maximum sales

4. **Sales Channel Performance**

   * Distribution across:

     * Direct
     * Online
     * Partner
     * Telesales

---

## 📈 Key Insights

* Partner/Online channels contribute significantly to total sales
* Certain industries (like Electronics/Retail) dominate revenue
* Top sales reps generate a major portion of sales
